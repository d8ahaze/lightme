Анализ и сбор факторов ранжирования

Каждый параметр представляет собой столбцы в базе. Поскольку доступ к DNS записям осуществляется владельцами сайта для каждого параметра реализуется своя логика накопления информации о нем. Если же пользователь предоставляет доступ, указывая наш ключ в TXT записи, то возможно использование api популярных поисковиков. Дальнейший анализ происходит по этим извлеченным и затем загруженным данным.
